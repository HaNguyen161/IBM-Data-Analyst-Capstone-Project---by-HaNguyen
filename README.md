## 📌 Project Overview

This repository contains my IBM Data Analyst Capstone Project, focused on analyzing **current technology adoption** and **future skill demand** using developer survey and job posting data.

The project applies a full data analysis workflow: data collection, data cleaning, exploratory data analysis, data visualization, dashboard development, and business insight generation. The final output identifies the most in-demand programming languages, databases, platforms, web frameworks, and workforce demographic patterns relevant to recruiters, learners, and career changers.

## 🎯 Business Objective

The main objective is to answer one key question:

> **Which technologies and skills should candidates prioritize to stay competitive in the technology job market?**

To answer this, the project investigates:

- Current technology usage among developers
- Future technology demand
- Popular programming languages, databases, platforms, and web frameworks
- Developer demographic patterns by age, country, and education level
- Practical implications for career planning and recruiter readiness

## 🔄 Project Workflow

### 1. Data Collection

Collected data from multiple sources including job posting datasets, API-based data, web scraping exercises, and the Stack Overflow Developer Survey dataset.

### 2. Data Cleaning & Preparation

Prepared the data for analysis by:

- Identifying and removing duplicate records
- Handling missing values
- Standardizing categorical variables
- Transforming multi-response survey fields
- Normalizing numerical fields when needed

### 3. Exploratory Data Analysis

Performed EDA to understand:

- Technology adoption patterns
- Distribution of compensation and job-related variables
- Outliers in numerical variables
- Relationships between selected fields
- Developer demographics and skill preferences

### 4. Data Visualization

- Bar charts
- Histograms
- Box plots
- Scatter plots
- Bubble plots
- Pie charts
- Dashboard-level comparative charts

### 5. Dashboard & Reporting

Built dashboard sections covering:

- Current technology usage
- Future technology trends
- Developer demographics
- Overall findings and implications

## 📊 Key Findings

### Programming Languages

Python is the strongest programming language across both current usage and future demand. Rust also shows strong future interest, while JavaScript appears to decline in future preference compared with current usage.

### Databases

PostgreSQL leads both current database adoption and future demand. Microsoft SQL Server, MySQL, and SQLite remain major database choices, while MongoDB shows continued relevance as a NoSQL option.

### Platforms & Web Frameworks

AWS stands out as a leading cloud platform, while React remains one of the most important web development frameworks. Cloud-related tools are increasingly important for data, backend, and full-stack roles.

### Demographics

The respondent base is concentrated in younger and early-to-mid career professionals, especially the 25–34 age group. The United States contributes the largest share of respondents, followed by other major technology markets. Bachelor’s and Master’s degrees are the most common education levels among respondents.

## 💡 Business Implications

- Candidates should build a balanced technology stack instead of relying on only one tool.
- Python, SQL/PostgreSQL, cloud platforms, and modern web frameworks are highly valuable for recruiter-ready profiles.
- Cloud and database skills are becoming essential across data analyst, backend, and full-stack career paths.
- Practical portfolio projects are important because they demonstrate the ability to connect coding, databases, dashboards, and business insights.
- Learners and career changers should prioritize job-ready skills that can be shown through real analysis projects.
